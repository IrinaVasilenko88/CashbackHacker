## Домашнее задание по курсу Автоматизированное тестирование

## Тема: Основы автоматизации

CashbackHacker - небольшой сервис, который сообщает пользователю о том, сколько ему нужно "докупить" в рамках конкретной траты, чтобы получить максимальное количество кэшбека.
